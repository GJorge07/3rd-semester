Jogo 2D - Trabalho de Programação 2 - 2026/1 - Gabriel Dobrowolski Jorge

Descrição

O jogador percorre uma floresta com obstáculos variados e uma parte subterrânea (caverna). O objetivo é chegar ao fim sem perder todas as vidas.

Obstáculos:

 Copo caindo
 Bola rolando
 Buraco (leva ao subsolo)
 Pássaros
 Morcego 
 Espinho
 Inimigo (Subsolo)

Estrutura dos arquivos:

 Arquivo | O que faz

 main.c        Entrada do programa 
 game.c/h      Loop principal, lógica e estados 
 player.c/h    Personagem, movimento e sprites 
 map.c/h       Cenário e backgrounds 
 obstacle.c/h  Obstáculos e animações 
 colision.c/h  Detecção de colisão
 hud.c/h       Interface, menus e vidas
